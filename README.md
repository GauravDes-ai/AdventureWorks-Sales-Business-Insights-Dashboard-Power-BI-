## 📌 Overview

This project presents a comprehensive **Business Intelligence dashboard** built using **Power BI** and the **AdventureWorks dataset**.

The dashboard enables stakeholders to monitor key business metrics, analyze sales performance, and uncover actionable insights related to products and customers.

---

## 🎯 Business Objectives

* Track revenue, orders, and returns against targets
* Identify top-performing products and categories
* Analyze customer behavior and segmentation
* Monitor return rates to detect potential issues
* Support data-driven strategic decisions

---

## 🛠️ Tools & Technologies

* Power BI Desktop
* Power Query (Data Transformation)
* DAX (Data Analysis Expressions)
* Data Modeling (Star Schema)
